# Imagens Sugeridas para o App ServiceGo

## Identidade visual
- Logo principal (fundo transparente): volante + rota + símbolo de check.
- Ícone simplificado: "SG" com estrada estilizada.
- Splash screen: gradiente azul petróleo -> azul escuro com carro em silhueta.

## Painel inicial
- Banner superior: foto urbana noturna com carro executivo (tons frios).
- Ilustração vazia para listas sem dados: motorista com prancheta.
- Ilustração de erro de conexão: ícone de nuvem/desconectado.

## Módulos
- Corridas: mini ícones para status (agendada, em andamento, concluída, cancelada).
- Financeiro: ícones de entrada (seta para cima) e saída (seta para baixo).
- Agenda: calendário com marcador e relógio.
- Clientes: avatar padrão neutro para contato sem foto.
- Veículos: conjunto de ícones lineares de sedã, SUV e van.

## Especificações recomendadas
- Banner painel: 1600x900 px.
- Splash: 1242x2436 px.
- Ícone app: 1024x1024 px.
- Ícones módulo: 128x128 px (SVG preferencial).

## Direção estética
- Paleta: azul petróleo, azul profundo, laranja de destaque.
- Estilo: profissional, limpo, sem excesso de detalhes.
- Evitar: cliparts genéricos, fundo roxo, imagens de baixa resolução.
